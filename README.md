# Weather Forecast App

A responsive weather forecast web application built using HTML, CSS, JavaScript, and the OpenWeather API.

## Features

- Search weather by city name
- Displays current temperature
- Displays minimum and maximum temperature
- Displays humidity
- Displays wind speed and wind direction
- Displays weather condition and description
- Responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API

## How to Use

1. Open the application.
2. Enter a city name.
3. Click the **Search** button.
4. View the current weather information.
